# S-Line-usb-camera
Simple preview for S_Line SC550 camera for Linux  writen in Python.  
  
For correctly working type in terminal:  
  
```shell  
lsusb  
```  
  
copy vendor before ':' to vid in code and copy parent after ':' to pid in code.  
  
Example: 
<p>ID: vendor:parent</p>
<p>ID: 1g3D:1032 </p>
